### About Me

Python backend developer building production microservices in a fintech context.
Day-to-day I design async APIs, message-driven services and the infrastructure
around them — from the ORM layer up to deployment.

- Core: **Python 3.12**, **FastAPI**, **Django** in a microservices architecture
- Data: **PostgreSQL**, **Redis**, **MongoDB** with **SQLAlchemy** / **Alembic**
- Async & messaging: **Celery**, **Kafka**, **RabbitMQ**, **gRPC**
- Quality & delivery: **Pytest**, **Docker**, **Nginx**, **CI/CD**
- Also comfortable in **Go** and **Bash** for tooling and automation
- Interested in **Machine Learning** and **Higher Mathematics**
